# USM v0.1.0 — Initial Release

This is the first public release of the Universal State Mirror (USM) server.

## Features
- Real-time WebSocket telemetry router
- JSON + binary frame support
- Multi-publisher, multi-viewer routing
- Pattern-based subscriptions (#, prefix, namespace)
- In-memory + on-disk storage
- Test publisher and viewer clients
- MIT License
- Professional README with badges

USM is now ready for robotics teams, simulation developers, and distributed systems engineers.
