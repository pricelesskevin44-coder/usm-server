# Contributing to USM

Thank you for your interest in contributing to the Universal State Mirror project!

## How to Contribute

### 1. Fork the repository
Create your own fork and clone it locally.

### 2. Create a feature branch
Use descriptive names:
- feature/add-binary-viewer
- fix/routing-bug
- docs/improve-readme

### 3. Follow the coding style
- Python 3.10+
- Keep modules small and focused
- Use async/await for all network operations

### 4. Write clear commit messages
Use:
- feat: for new features  
- fix: for bug fixes  
- docs: for documentation  
- refactor: for internal changes  

### 5. Submit a pull request
Explain:
- What you changed  
- Why it matters  
- How to test it  

We welcome contributions from robotics engineers, researchers, and developers.
